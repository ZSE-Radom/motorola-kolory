Użytkowanie

Kolory ZSE to narzędzie umożliwiające eksplorację różnorodnych harmonii kolorystycznych na podstawie wybranego koloru bazowego. Użytkownik może wybrać kolor za pomocą suwaków dla składowych HSL (Odcień, Nasycenie, Jasność) albo za pomocą systemowego selektora kolorów. Po wprowadzeniu koloru, program generuje różnorodne harmonie kolorów, które są wyświetlane w interaktywnym interfejsie użytkownika.

Jak używać Kolory ZSE?

  1. Wybierz kolor bazowy za pomocą suwaków HSL lub systemowego selektora kolorów.
  2. Program automatycznie generuje różnorodne harmonie kolorów na podstawie wybranego koloru bazowego.
  3. Kliknij na przycisk "Kopiuj" przy harmonii kolorów, aby skopiować jej wartości do schowka.
  4. Możesz również wyświetlić podgląd wybranych harmonii kolorów.
  5. Przeglądaj różnorodne harmonie kolorów, eksploruj ich składowe HSL oraz inne systemy kolorów, takie jak RGB, CMYK, XYZ, LAB, itp.

Funkcje dodatkowe

Poza eksploracją harmonii kolorów, program Kolory ZSE oferuje funkcje dodatkowe, takie jak eksportowanie harmonii kolorów do pliku JSON, importowanie harmonii kolorów z pliku JSON. Json składa sie z listy kolorów reprezentowanych w różnych systemach barw.
